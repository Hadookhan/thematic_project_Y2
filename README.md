# thematic_project_Y2
Second year CS thematic project. Team of 6 working to create a 'Movie Database Presentation Tool'.

## System brief
A web-driven system that displays movie information based on various features given via the IMDB database.

## Run on local machine
```
pnpm install
pnpm dev
```

### Prerequisites:
- Node ~ latest
