# thematic_project_Y2
Second year CS thematic project. Team of 6 working to create a 'Mode Database Presentation Tool'.

## System brief
<p>
    A web-driven system that displays movie information based on various features given via the IMDB database.
</p>

## Run on local machine
```
    npm run dev
```

### Prerequisites:
    - Node ~ latest
